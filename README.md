# Камери МК

https://kameri-mk.herokuapp.com/  

## Инфо
RAM: ~ 4 GB (мин: 3 GB)  
NET: ~ 1 Mbps (мин: 0.5 Mbps)  

## Упатство

1. Отвори го линкот
2. F11
3. Кликни на видеото
4. F за да го видиш видеото на цел екран
5. Esc за да се вратиш назад
